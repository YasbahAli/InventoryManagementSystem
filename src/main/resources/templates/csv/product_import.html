CSV Format Requirements

Required Headers (in order):

Name,Description,Quantity,Price,Category

Example:

Laptop,High performance laptop,10,999.99,Electronics
Mouse,Wireless mouse,50,29.99,Electronics
Desk,Wooden desk,5,299.99,Furniture

Notes:
- Description is optional (can be empty)
- Quantity and Price are required numbers
- Category should match existing categories in the system
- If category doesn't exist, product will be created without a category

Only .csv files are accepted
