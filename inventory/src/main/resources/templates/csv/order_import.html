CSV Format Requirements

Required Headers (in order):

Product,Quantity,Status,Supplier

Example:

Laptop,5,PENDING,TechSuppliers
Mouse,20,CONFIRMED,OfficeGoods
Desk,2,PENDING,

Notes:
- Product name is required and must exist in the system
- Quantity is required (must be > 0)
- Status is optional. Valid values: PENDING, CONFIRMED, SHIPPED, COMPLETED, CANCELLED
- Default status is PENDING if not specified
- Supplier is optional (leave blank for none)
- Supplier name must match existing suppliers in the system

Only .csv files are accepted
